# Makina Elemanları Hesaplayıcısı

Mühendislik öğrencileri için makina elemanları hesaplama aracı.

## Özellikler

- **Normal Modül Hesabı** — Diş dibi mukavemetine göre normal modül hesabı (form faktörü interpolasyonu dahil)
- **Mil Hesabı** — Standart mil çaplarına göre mil ucu uzunluğu ve tolerans seçimi
- **Kuvvet Hesabı** — Md, β₀ ve taksimat çapına göre Ft, Fr, Fe hesapları (2 kademe)
